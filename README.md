# ЛИЧНЫЙ САЙТ
## Базовое описание
Личный сайт с биографией, резюме и реализованными проектами. Написан на React, дизайн создан в Figma. Интегрирован с MongoDB, где хранятся статьи. Серверная часть: Node JS + Express.
